# App para Presupuestos

## Descripción

Esta es una aplicación web para presupuestos que permite a los usuarios crear, gestionar y visualizar sus presupuestos de manera sencilla y efectiva. Utilizando HTML, CSS y JavaScript, esta app proporciona una interfaz amigable y funcional que facilita la planificación financiera personal.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura del contenido.
- **CSS3**: Para el diseño y la presentación.
- **JavaScript**: Para la interactividad y la funcionalidad dinámica.

## Funcionalidades

- Crear nuevos presupuestos con categorías y montos.
- Editar y eliminar presupuestos existentes.
- Visualizar un resumen de gastos y presupuestos.
- Interfaz responsiva que se adapta a diferentes dispositivos.
